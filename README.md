# 🎤 AI Voice Assistant

A voice-enabled AI chatbot that listens, thinks, and speaks.

## 🚀 Features
- 🎤 Speech-to-text (voice input)
- 🤖 AI response using OpenRouter
- 🔊 Text-to-speech (voice output)
- 🔁 Continuous conversation
- 🛑 Stop & reset controls

## 🛠️ Tech Stack
- FastAPI (Backend)
- HTML, JavaScript (Frontend)
- OpenRouter API (LLM)
- Web Speech API

## 📦 How to Run

### Backend

uvicorn app:app --reload --port 8001
